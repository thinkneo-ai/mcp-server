# ThinkNEO Control Plane — MCP Server

## Overview
ThinkNEO is an Enterprise AI Control Plane that sits between your applications and AI providers, enforcing governance on every request.

## Capabilities
- Runtime guardrails (input/output/context/tool-use controls in monitor and enforce modes)
- Deep observability and immutable audit logs (SIEM-ready exports)
- AI FinOps: department-level cost attribution and hard budget enforcement
- Agent lifecycle control with approval workflows and traceable action records
- Compliance readiness: RBAC, tenant isolation, evidence workflows
- Provider agnostic: OpenAI, Anthropic, Google, NVIDIA NIM, Mistral, DeepSeek, and more

## Remote Endpoint
